# 📱 ПОЛНАЯ ИНСТРУКЦИЯ ПО ЗАПУСКУ — АудиоКнига

Этот гайд написан для человека, который **никогда раньше не делал мобильные приложения**.
Следуйте строго по шагам, не пропускайте ни одного.

---

## ШАГ 1 — Установите Node.js на компьютер

Node.js — это программа, без которой ничего не запустится.

1. Откройте браузер и перейдите на сайт: **https://nodejs.org**
2. Нажмите на большую зелёную кнопку **"LTS"** (это стабильная версия)
3. Скачайте и установите как обычную программу (просто нажимайте "Далее")
4. После установки **перезагрузите компьютер**

**Проверка:**
- Откройте «Командную строку» (Windows: нажмите Win+R, напишите `cmd`, Enter)
- Напишите: `node --version` и нажмите Enter
- Должно показать что-то вроде: `v20.11.0` ✅

---

## ШАГ 2 — Установите Git

Git нужен для скачивания зависимостей.

1. Перейдите на: **https://git-scm.com/downloads**
2. Скачайте версию для Windows и установите (везде нажимайте "Next")

---

## ШАГ 3 — Установите Expo CLI

Expo — это инструмент, который позволяет запускать приложение прямо на телефоне без сборки.

Откройте командную строку и выполните:
```
npm install -g expo-cli eas-cli
```
Подождите пока установится (1-2 минуты).

---

## ШАГ 4 — Распакуйте проект

1. Скачанный архив `audiobook-expo.zip` распакуйте в папку
   Например: `C:\Projects\audiobook-expo\`
2. Откройте командную строку **в этой папке**:
   - В проводнике перейдите в папку
   - В адресной строке проводника напишите `cmd` и нажмите Enter
   - Откроется командная строка уже в нужной папке ✅

---

## ШАГ 5 — Установите все зависимости проекта

В командной строке (в папке проекта) выполните:
```
npm install
```
Это скачает все необходимые библиотеки. Займёт 3-5 минут.
Не пугайтесь предупреждений (warnings) — это нормально.

---

## ШАГ 6 — Создайте бесплатный аккаунт Supabase (база данных)

Supabase — это бесплатная база данных в облаке. Там будут храниться пользователи, книги и прогресс.

1. Перейдите на: **https://supabase.com**
2. Нажмите **"Start your project"**
3. Войдите через GitHub (нужно создать аккаунт GitHub, если нет — это бесплатно на github.com)
4. Нажмите **"New project"**
5. Заполните:
   - **Name**: audiobook (любое название)
   - **Database Password**: придумайте пароль и **запишите его**
   - **Region**: выберите ближайший (например, Frankfurt или London)
6. Нажмите **"Create new project"** и подождите 1-2 минуты

---

## ШАГ 7 — Настройте базу данных

1. В панели Supabase слева нажмите **"SQL Editor"**
2. Нажмите кнопку **"New query"**
3. Откройте файл `supabase_schema.sql` из папки проекта (любым текстовым редактором)
4. Выделите **весь текст** (Ctrl+A) и скопируйте (Ctrl+C)
5. Вставьте в поле SQL Editor (Ctrl+V)
6. Нажмите кнопку **"Run"** (или Ctrl+Enter)
7. Должно появиться сообщение "Success" ✅

---

## ШАГ 8 — Получите ключи Supabase

1. В панели Supabase слева нажмите **"Settings"** (шестерёнка внизу)
2. Нажмите **"API"**
3. Вы увидите два значения — скопируйте их:
   - **Project URL** — выглядит как `https://abcdefghijk.supabase.co`
   - **anon public** — длинная строка, начинается с `eyJ...`

---

## ШАГ 9 — Вставьте ключи в проект

1. Откройте файл `src/services/supabase.ts` в текстовом редакторе
   (можно открыть через Блокнот, или лучше скачайте **VS Code** — бесплатный редактор кода: code.visualstudio.com)
2. Найдите эти строки:
   ```
   export const SUPABASE_URL = 'https://ВАША_ССЫЛКА.supabase.co';
   export const SUPABASE_ANON_KEY = 'ВАШ_ANON_KEY';
   ```
3. Замените значения на ваши (именно URL и ключ в кавычках):
   ```
   export const SUPABASE_URL = 'https://abcdefghijk.supabase.co';
   export const SUPABASE_ANON_KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...';
   ```
4. Сохраните файл (Ctrl+S)

## ШАГ 10 — Установите приложение Expo Go на телефон

**На Android телефоне:**
1. Откройте Google Play Store
2. Найдите **"Expo Go"**
3. Установите приложение

**Важно:** телефон и компьютер должны быть подключены к **одной Wi-Fi сети**!

---

## ШАГ 11 — Запустите приложение

В командной строке (в папке проекта) выполните:
```
npx expo start
```

Подождите 30-60 секунд. Должен появиться **QR-код** в командной строке.

**На Android:**
1. Откройте приложение **Expo Go** на телефоне
2. Нажмите **"Scan QR code"**
3. Наведите камеру на QR-код в командной строке
4. Приложение загрузится на телефоне ✅

---

## ШАГ 12 — Добавьте тестовые книги

Чтобы проверить, что всё работает, добавьте книгу с реальным аудио:

1. В Supabase → **"Table Editor"**
2. Выберите таблицу **"books"**
3. Нажмите **"Insert row"** и заполните:
   - `title`: Мастер и Маргарита
   - `author`: Булгаков М
   - `narrator`: Самойлов В.
   - `is_popular`: true
   - `total_chapters`: 1
4. Запишите `id` этой книги (нажмите на строку)
5. Перейдите в таблицу **"chapters"**
6. Нажмите **"Insert row"**:
   - `book_id`: вставьте id книги
   - `chapter_number`: 1
   - `title`: Глава 1
   - `audio_url`: ссылка на MP3 файл (например, загрузите на Google Drive и возьмите прямую ссылку)
   - `duration`: 3600 (секунды)

---

## ШАГ 13 — Добавление аудиофайлов (как загружать книги)

### Вариант А — Supabase Storage (рекомендую):
1. В Supabase слева нажмите **"Storage"**
2. Нажмите **"New bucket"**, назовите `audio`, поставьте **Public**
3. Загрузите MP3 файл
4. Нажмите на файл → **"Get URL"** — скопируйте эту ссылку
5. Вставьте в поле `audio_url` в таблице `chapters`

### Вариант Б — Любое облако:
Можно использовать любую прямую ссылку на MP3:
- Google Drive (нужна прямая ссылка для скачивания)
- Яндекс Диск
- Любой CDN

## ШАГ 14 — Сборка APK файла (для распространения)

Когда захотите установить приложение без Expo Go (готовый APK):

1. Зарегистрируйтесь на: **https://expo.dev** (бесплатно)
2. В командной строке выполните:
```
eas login
```
3. Введите ваш email и пароль от expo.dev
4. Затем:
```
eas build --platform android --profile preview
```
5. Ответьте "yes" на все вопросы
6. Дождитесь сборки (15-20 минут)
7. Получите ссылку на скачивание APK файла
8. Откройте ссылку на телефоне и установите APK

## ❓ ЧАСТЫЕ ПРОБЛЕМЫ

**"command not found: npm"**
→ Node.js не установлен или нужна перезагрузка компьютера

**"Cannot connect to server"**
→ Проверьте что SUPABASE_URL и SUPABASE_ANON_KEY вставлены правильно

**QR-код сканируется но приложение не открывается**
→ Убедитесь что телефон и компьютер в одной Wi-Fi сети

**"Network request failed" при входе**
→ Проверьте интернет-соединение и ключи Supabase

**Пустой список книг**
→ Добавьте книги через Supabase Table Editor (ШАГ 12)

**TalkBack не читает элементы**
→ Включите TalkBack: Настройки → Специальные возможности → TalkBack → Включить
