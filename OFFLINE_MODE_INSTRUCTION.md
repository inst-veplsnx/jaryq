# 📴 Офлайн-режим — Инструкция

JARYQ теперь поддерживает скачивание аудиокниг для прослушивания **без подключения к интернету**. Идеально для метро, поездок и мест с плохим соединением.

---

## 🎯 Возможности

| Функция | Описание |
|---|---|
| 📥 **Скачивание главы** | Каждая глава может быть загружена отдельно |
| 📚 **Скачать все главы** | Одним нажаттием — все главы книги |
| ✅ **Индикатор статуса** | Видно, какие главы доступны офлайн |
| 📊 **Прогресс загрузки** | Отображается процент скачивания |
| 🗑️ **Управление хранилищем** | Удаление загрузок для освобождения места |
| 🎧 **Автоматическое воспроизведение** | Загруженные главы играют из локального кэша |

---

## 📱 Как скачать главу

### Из страницы книги:

1. Откройте страницу книги
2. В списке глав нажмите 📥 рядом с нужной главой
3. Дождитесь завершения загрузки (появится ✅)
4. Глава теперь доступна офлайн

### Из плеера:

1. Откройте плеер во время воспроизведения
2. Нажмите кнопку 📥 внизу экрана
3. Дождитесь завершения загрузки
4. Появится индикатор "📴 Офлайн"

---

## 📚 Как скачать всю книгу

1. Откройте страницу книги
2. Нажмите **"Скачать все главы (N)"**
3. Главы скачаются последовательно
4. Уже загруженные главы будут пропущены
5. После завершения все главы доступны офлайн

---

## 🔄 Управление загрузками

### Удалить одну главу:
- Нажмите ✅ рядом с загруженной главой
- Подтвердите удаление

### Удалить все главы книги:
- Если все главы загружены, нажмите кнопку "Все главы загружены"
- Подтвердите удаление

### Удалить все загрузки:
1. Откройте **Профиль** → **Настройки**
2. Найдите секцию **"Офлайн загрузки"**
3. Нажмите **"Удалить все загрузки"**

---

## 💾 Хранилище

### Где хранятся файлы?
Файлы сохраняются во внутреннем хранилище приложения:
```
<documentDirectory>/audio/<bookId>/<chapterId>.m4a
```

### Сколько места занимает книга?
Зависит от количества глав и их длительности:
- **1 глава (1 час)** ≈ 30-50 MB (при 64kbps AAC)
- **Книга (10 глав)** ≈ 300-500 MB
- **Книга (20 глав)** ≈ 600 MB - 1 GB

### Как узнать размер загрузок?
Откройте **Профиль** → **Настройки** → **Офлайн загрузки**
Там указано количество книг и общий размер.

## 🎧 Воспроизведение офлайн

### Автоматический режим
- Если глава загружена, она **автоматически** воспроизводится из локального файла
- Если глава не загружена, воспроизводится из интернета
- Индикатор "📴 Офлайн" показывает доступность офлайн-версии

### Переключение онлайн/офлайн
Приложение автоматически определяет доступность локального файла:
- ✅ Загруженная глава → играет из локального кэша
- 🌐 Не загруженная глава → играет из интернета

|---|---|
| 📥 | Глава не загружена, нажмите для скачивания |
| ⬇️ 45% | Глава скачивается (45% завершено) |
| ✅ | Глава загружена и доступна офлайн |

## ⚙️ Технические детали

### Используемые технологии
- **expo-file-system** — скачивание и управление файлами
- **expo-av** — воспроизведение локальных файлов
- **Zustand** — управление состоянием загрузок

### Формат аудио
Рекомендуется **AAC (.m4a)** с битрейтом 64-96 kbps:
- Отличное качество для аудиокниг
- Компактный размер файлов
- Поддержка перемотки и Range requests

### Резюмеемость загрузок
Загрузки сохраняются между сессиями. При перезапуске приложения:
- Все загруженные файлы обнаруживаются автоматически
- Состояние загрузок восстанавливается

## 🔧 Устранение проблем

### Загрузка не начинается
**Причина:** Нет интернета или проблема с сервером

**Решение:**
1. Проверьте подключение к интернету
2. Убедитесь, что URL аудиофайла доступен
3. Попробуйте снова через некоторое время

### Загрузка зависла
**Причина:** Обрыв соединения или ошибка сервера

**Решение:**
1. Подождите несколько секунд
2. Если не помогло — удалите и начните заново
3. Проверьте стабильность интернета

### "Недостаточно места"
**Причина:** Переполнено хранилище устройства

**Решение:**
1. Удалите ненужные загрузки
2. Освободите место на устройстве
3. Попробуйте снова

### Файл не воспроизводится офлайн
**Причина:** Файл поврежден или не полностью загружен

**Решение:**
1. Удалите файл и скачайте снова
2. Проверьте стабильность интернета при загрузке
3. Убедитесь, что файл в формате .m4a или .mp3

## 💡 Советы

### 1. Скачивайте заранее
Скачайте книги дома перед выходом, чтобы слушать в метро без расхода трафика.

### 2. Используйте Wi-Fi
Скачивайте книги только через Wi-Fi для экономии мобильного трафика.

### 3. Следите за хранилищем
Регулярно удаляйте прослушанные книги для освобождения места.

### 4. Проверяйте загрузки перед поездкой
Откройте книгу и убедитесь, что все нужные главы имеют ✅.

### 5. Приоритет глав
Если мало места, скачайте только следующие 2-3 главы, которые планируете слушать.
